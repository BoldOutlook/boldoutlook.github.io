Bedrock

Full-featured, Lightweight Sass Front-end Framework.

Bedrock is a lightweight, feature-rich front-end framework for developing responsive, mobile first websites and web apps. Bedrock is built with the Sass stylesheet language and Grunt, a JavaScript task runner.

Download (zip) Download (tar.gz) Style Guide

Features

Bedrock is only 14KB when minified and 20KB unminified.

Responsive and mobile-first design.

CSS for typography, code, buttons, forms, images and tables.

CSS for social sharing buttons, icons and links.

Current components include: alerts.

Documentation

The following documentation will help you get started building websites and web apps using the Bedrock front-end framework.

Installation

Install using Git:

git clone https://github.com/BoldOutlook/Bedrock

Install using Bower:

bower install get-bedrock

Adding to your website

You can add Bedrock to your website using the following HTML code:

<link rel="stylesheet" type="text/css" href="path/bedrock.min.css" />

Make sure you change path/ with the path of the Bedrock CSS file.
